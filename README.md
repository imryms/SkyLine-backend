### ***Description***
#### A web platform for donating medical equipment. Users can submit items with condition details. Admins review submissions, and approved items are displayed for those in need.
***

### ***Technologies Used***
* Node.js
* Express.js
* MongoDB
* EJS
* JavaScript
* JSON
